# Cyborg Cassowary

<img alt="Cyborg Cassowary" src="http://www.cyborgcassowary.com/wp-content/themes/stitch-child-01/screenshot.png" style="display:block; margin:0 auto; width:100%; max-width:650px; height:auto; border:#222 solid 1px;" />

[Cyborg Cassowary](http://www.cyborgcassowary.com) is a record keeper for Ryan Hettler's and Kate Miterkó's jam session ideas. Jam sessions also include the occasional covers. 

Cyborg Cassowary is built on Wordpress with a child theme of [Stitch](https://wordpress.org/themes/stitch) and uses [Timber](https://github.com/jarednova/timber/) for theme modifications.
